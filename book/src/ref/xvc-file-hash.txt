xvc-file-hash 0.2.6
Calculate hash of given files

Note that this doesn't use .xvcignore facility and doesn't require an xvc root. It loads the
configuration from xvc repository if it runs within, otherwise uses user, system or default options.

USAGE:
    xvc file hash [OPTIONS] [TARGETS]...

ARGS:
    <TARGETS>...
            Files to process

OPTIONS:
    -a, --algorithm <ALGORITHM>
            Algorithm to calculate the hash. One of blake3, blake2, sha2, sha3. All algorithm
            variants produce 32-bytes digest

    -h, --help
            Print help information

        --text-file
            Consider the file as a text file. Otherwise uses [is_text_file] function to decide

    -V, --version
            Print version information
