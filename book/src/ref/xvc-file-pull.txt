xvc-file-pull 
Fetch files from remote

USAGE:
    xvc file pull [OPTIONS] --remote <REMOTE> [TARGETS]...

ARGS:
    <TARGETS>...    targets to push to remote

OPTIONS:
        --force              force even if the files are already present in the workspace
    -h, --help               Print help information
    -r, --remote <REMOTE>    remote name or guid to send the files
