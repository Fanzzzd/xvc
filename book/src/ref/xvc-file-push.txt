xvc-file-push 
Push files to remote

USAGE:
    xvc file push [OPTIONS] --remote <REMOTE> [TARGETS]...

ARGS:
    <TARGETS>...    targets to push to remote

OPTIONS:
        --force              force even if the files are already present
    -h, --help               Print help information
    -r, --remote <REMOTE>    remote name or guid to send the files
