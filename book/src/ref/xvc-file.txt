xvc-file 
File and directory management commands

USAGE:
    xvc file [OPTIONS] <SUBCOMMAND>

OPTIONS:
    -c, --config <CONFIG>      Configuration options set from the command line in the form
                               section.key=value
    -C <WORKDIR>               [default: .]
    -h, --help                 Print help information
        --no-env-config        Ignore configuration options from the environment
        --no-local-config      Ignore local config (.xvc/config.local)
        --no-project-config    Ignore project config (.xvc/config)
        --no-system-config     Ignore system config
        --no-user-config       Ignore user config
        --quiet                Suppress error messages
    -v, --verbose              Verbosity level, use multiple times to increase

SUBCOMMANDS:
    checkout    Add to track files using XVC
    fetch       Fetch files from remote
    hash        Calculate hash of given files
    help        Print this message or the help of the given subcommand(s)
    list        List files tracked with XVC 
    pull        Fetch files from remote
    push        Push files to remote
    track       Track file versions using XVC
