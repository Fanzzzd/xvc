xvc-init 
Initialize an Xvc project

USAGE:
    xvc init [OPTIONS]

OPTIONS:
        --force          Create the repository even if already initialized. Overwrites the current
                         .xvc directory Resets all data and guid, etc
    -h, --help           Print help information
        --no-git         Don't require Git
        --path <PATH>    Path to the directory to be intialized. (default: current directory)
