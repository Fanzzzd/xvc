xvc-pipeline-import 
Import the pipeline from a file

USAGE:
    xvc pipeline import [OPTIONS]

OPTIONS:
        --file <FILE>        File to read the pipeline. Reads from stdin if not specified.
        --format <FORMAT>    Format for input from stdin.
    -h, --help               Print help information
    -n, --name <NAME>        Name of the pipeline this command applies to
        --overwrite          Whether to overwrite the current pipeline if one found with an
                             identical name
