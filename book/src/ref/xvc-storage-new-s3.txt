xvc-storage-new-s3 
Add a new S3 remote

USAGE:
    xvc storage new s3 [OPTIONS] --name <NAME> --bucket-name <BUCKET_NAME> --region <REGION>

OPTIONS:
        --bucket-name <BUCKET_NAME>
            S3 bucket name

    -h, --help
            Print help information

    -n, --name <NAME>
            Name of the remote
            
            This must be unique among all remotes of the project

        --region <REGION>
            AWS region

        --remote-prefix <REMOTE_PREFIX>
            You can set a directory in the bucket with this prefix
            
            [default: ]
